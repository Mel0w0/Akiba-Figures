Proyecto Alura Akiba Figures
Introducción 
Este es un proyecto de una tienda online con su apartado de productos habilitados en la API y un
formulario para subir más, con este proyecto propuesto por Alura Latam se tiene la intención de
practicar los temas vistos en los diferentes cursos, como el manejo del DOM, el consumir y manejar
datos de una API, la creación de solicitudes con los métodos HTTP. 

Tecnologías utilizadas 

• HTML,CSS y Javascript
• Herramientas de desarrollo: Visual Studio Code
• Deploy: GitHub pages

Características 

•Diseño moderno y responsivo.
•Listado de figuras anime y de acción de diversas tiendas online referenciadas en su descripción.
•Formulario para subir más figuras.
•Compatible con navegadores modernos.

Instalación
Este proyecto se puede descargar desde su propio repositorio en Github, dando clic al botón verde 
que dice "Code" y luego dando a la opción "Descargar ZIP"
